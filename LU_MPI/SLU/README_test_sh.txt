Аналог скрипта Бгч...

Положите его там же где весь код.
В самом скрипте укажите относительный путь до тестовых файликов:
	dir="../tests"

Вывод у него в двух режимах:
# чтобы получить весь вывод вашей проги: 
	bash test.sh

# чтобы выводились ток время и невязка: 
	bash test.sh 1

Удобно еще перенаправить вывод в файл >res.txt

Ещё если сообщение об ошибках начаинается с "ERROR", то второй режим выведет и эту строку.
